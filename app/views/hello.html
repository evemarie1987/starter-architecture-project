project architecture

divide every piece of design as an independant “module” in : root/css/module/my-module.less
that way you can easily maintain & reuse your code across other projects

define project specificities in “variables” in : root/css/base/vars
define project base design in root/css/base/base

code every css animation in : root/css/animation/keyframes
divide every animation in an specific file : root/css/animation/
my-animation
